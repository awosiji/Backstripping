## How to Contribute

### 1. Reporting Issues
If you find a bug or have a suggestion:
- Open an issue in the **Issues** tab.
- Include a clear description and, if possible, an example or screenshot.

### 2. Improving the Notebook
You can contribute by:
- Adding better documentation or comments.
- Cleaning up code cells.
- Improving figures or adding examples.
- Adding new stratigraphic datasets.

### 3. Pull Requests
If you'd like to make a code or documentation change:
1. Fork the repository  
2. Create a new branch  
3. Make your changes  
4. Submit a pull request  

Please ensure your changes:
- Are clearly documented  
- Include before/after explanations if modifying plots  
- Do not commit notebook checkpoint files or cache files  

### 4. Style Guidelines
- Use clear variable names  
- Combine cells when appropriate  
- Avoid committing temporary files or images
